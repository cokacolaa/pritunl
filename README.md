### Install Pritunl VPN
```
git clone https://github.com/dylansysops/pritunl.git && cd ./pritunl && chmod +x ./openvpn-intsall.sh && ./openvpn-intsall.sh -y
```

### Login WEB UI

http://ip_host

- Setup Key
```
sudo pritunl setup-key 
```

- Lấy username và password
```
sudo pritunl default-password 
```
